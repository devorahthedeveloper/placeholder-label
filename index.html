Placeholder label

Placeholder label is a custom placeholder polyfill I wrote for use on the FXCM websites. It uses the label element as stand-in for the placeholder and works by absolutely positioning each label over its associated input, showing and hiding the label at the appropriate time.

Native placeholder support

You can override native placeholder support and force placeholder-label to invoke for all situations by setting overrideNative to true. By default, however, overrideNative is set to false and the plugin will exit silently if it detects native placeholder support.

Select menus

This plugin also includes optional support for <select> menus by dynamically inserting an option first-child whose text is that of the label element.

Usage scenarios

Since this plugin was designed specifically for use within the FXCM ecosystem, it relies upon specific DOM structure (the label and input must either be direct siblings or wrapped with a specified wrapper element).
